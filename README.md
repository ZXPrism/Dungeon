# Dungeon

A simple dungeon game written in python, since `2026.05.15`.

# Getting Started
1. Clone this repo
2. Run `uv sync` (install uv first)
3. Activate venv
4. Run `python -m dungeon.main`

This is a bit tedious. I will support packing codes into executable in the future.

# Features
- A mixture of classic game elements
  - You may find something familiar...
- Basic ECS
  - I love data driven!
